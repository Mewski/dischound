# Dischound

Discord friend graph visualizer. Paste the collector script in Discord's console, refresh, and see your friend network.

## Quick start

```sh
bun install
bun run dev
```

Open `localhost:5173`, click **Collect new data**, paste the script in Discord (F12 > Console), and refresh when done.

## Build

```sh
bun run build
```
